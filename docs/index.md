## 项目概述

CourseVideoGen 是一个 PPT 讲解视频自动生成工具，可以：

- 从 PPTX 导入幻灯片
- 使用 HTML 幻灯片（JSON 配置）
- 自动生成讲解音频（Edge-TTS）
- 合成图片+音频为视频
- 管理多个项目
