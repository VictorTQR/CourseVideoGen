# CourseVideoGen - PPT 讲解视频自动生成工具

**版本：** v1.0  
**日期：** 2026-04-26  
**作者：** ArkClaw

---

## 1. 产品概述

### 1.1 产品定位
一款自动化工具，帮助用户将 PPT 演示文稿转换为带有讲解音频的教学/演示视频。

### 1.2 目标用户
- 教师、培训师
- 内容创作者
- 需要制作教程视频的职场人士

### 1.3 解决的痛点
- PPT 录屏麻烦且不清晰
- 后期配音耗时耗力
- 需要专业的视频编辑技能

---

## 2. 功能需求

### 2.1 核心功能

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|--------|
| **PPT 解析** | 支持将 .pptx 文件解析为每页一张图片 | P0 |
| **讲解稿管理** | 用户可为每页 PPT 输入/编辑讲解文本 | P0 |
| **语音生成** | 使用 TTS 自动为每页讲解稿生成音频 | P0 |
| **视频合成** | 将 PPT 图片 + 音频合成为最终视频 | P0 |
| **项目管理** | 保存和加载项目，支持继续编辑 | P1 |

## 3. 产品流程

### 3.1 主流程图

```
┌─────────────────┐
│  1. 导入 PPT    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  2. PPT→图片    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  3. 输入讲解稿  │  ← 用户输入
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  4. 生成音频    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  5. 生成视频    │
└────────┬────────┘
         │
         ▼
┌─────────────────┐
│  6. 合并音视频  │
└─────────────────┘
```

### 3.2 详细说明

1. **PPT 解析**
   - 输入：.pptx 文件
   - 输出：slide_01.png, slide_02.png, ...

2. **讲解稿管理**
   - 支持导入 txt/markdown 文件
   - 或在界面中逐页编辑

3. **音频生成**
   - 使用 Edge-TTS
   - 每页生成独立的 .mp3 文件
   - 支持选择发音人、语速

4. **视频生成**
   - 图片 + 对应音频时长
   - 生成无音轨的 .mp4
   - 支持转场效果（可选）

5. **音视频合并**
   - 将所有音频拼接
   - 混流到视频中

## 4. 技术方案

### 4.1 技术选型

| 模块 | 技术方案 | 说明 |
|------|---------|------|
| PPT 解析 | `python-pptx` | 提取文本和布局 |
| PPT→图片 | `pywin32` (Windows) / `unoconv` (Linux) / `LibreOffice` | 渲染 PPT 为图片 |
| TTS | `edge-tts` | 微软语音服务 |
| 视频处理 | `moviepy` / `ffmpeg-python` | 视频合成与处理 |
| 界面 | `Streamlit` / `Gradio` (可选) | Web UI |

## 5. 用户使用流程

### 5.1 命令行使用（MVP）

```bash
# 1. 创建新项目
python main.py create "我的课程"

# 2. 导入 PPT
python main.py import-ppt "lesson.pptx"

# 3. 编辑讲解稿（或手动编辑 workspace/.../scripts/）
python main.py edit-script

# 4. 生成音频
python main.py generate-audio

# 5. 生成视频
python main.py generate-video

# 6. 一键全部生成
python main.py run-all
```

### 5.2 界面使用（v2.0，可选）

- 上传 PPT
- 可视化编辑每页讲解稿
- 试听音频
- 导出视频

## 7. 风险与依赖

| 风险 | 应对方案 |
|------|---------|
| PPT 渲染在 Linux 下不稳定 | 提供多种渲染方案备选 |
